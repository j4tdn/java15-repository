1. What is primary key of Result table.
Primary key of Result table is student_id and subject

2. List name, gender and teacher name of all students.
SELECT Student.name, Student.gender, Class.teacher
FROM Student 
JOIN Class ON Student.class_id = Class.id;

3. List teacher name and number of all student in class 12C.
SELECT Class.teacher, count(Student.id)
FROM Student 
JOIN Class ON Student.class_id = Class.id
WHERE Class.name LIKE '%12C%';

4.List the class name and number of the students who have good at Math (score>=8) and good 
at Literature (score>=8).

SELECT Class.name, count(Student.id) as HocSinhGioiToanVan
FROM Student 
JOIN Class ON Student.class_id = Class.id
JOIN Result ON Result.student_id = Student.id
GROUP BY Class.id
HAVING (Result.subject = 'Math' OR Result.subject = 'Literature') AND Result.score >= 8;
